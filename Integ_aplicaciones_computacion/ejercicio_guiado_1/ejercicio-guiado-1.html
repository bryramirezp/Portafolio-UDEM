Ejercicio Guiado 1

Clasificador de Servicios Cloud con IA

1. Objetivo del Proyecto

Desarrollar una herramienta CLI en Python que clasifique automáticamente servicios de cloud computing en las categorías IaaS, PaaS, SaaS y FaaS utilizando inteligencia artificial.

Características principales:

  • Interfaz de línea de comandos intuitiva
  • Clasificación automática usando IA
  • Resultados en formato JSON
  • Modo interactivo y batch
2. Tecnologías Utilizadas

Backend

  • Python 3.x - Lenguaje principal
  • OpenAI API - Clasificación con IA
  • Click - Framework CLI
  • Pydantic - Validación de datos

Herramientas

  • Docker - Containerización
  • Git - Control de versiones
  • Poetry - Gestión de dependencias
  • Pytest - Testing

4. Implementación Clave

Clasificador Principal

class CloudServiceClassifier:
    def __init__(self, api_key: str):
        self.client = OpenAI(api_key=api_key)
    
    def classify_service(self, service_name: str, description: str) -> CloudService:
        prompt = f"""
        Clasifica el siguiente servicio de cloud computing:
        Nombre: {service_name}
        Descripción: {description}
        
        Categorías: IaaS, PaaS, SaaS, FaaS
        """
        
        response = self.client.chat.completions.create(
            model="gpt-3.5-turbo",
            messages=[{"role": "user", "content": prompt}]
        )
        
        return self._parse_response(response)

6. Lecciones Aprendidas

Desafíos

  • Optimización de prompts para IA
  • Manejo de límites de API
  • Validación de respuestas inconsistentes

Soluciones

  • Prompts estructurados y específicos
  • Implementación de rate limiting
  • Validación robusta con Pydantic
